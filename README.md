## Overview

After completing the **AWS Academy Cloud Architecting track**, this project demonstrates how **AI model training** and **Generative AI labs** were successfully integrated into community college cloud computing courses using:

- **Amazon Bedrock**
- **Amazon SageMaker**

This initiative bridges the gap between **professional AWS training** and **classroom implementation**, enabling students to gain hands-on experience with modern AI systems.

---

## Purpose

This repository documents a **practical, replicable framework** for integrating applied AI into cloud curricula, including:

- Curriculum redesign aligned with AWS services  
- Lab environment architecture  
- Security and IAM considerations  
- Student project design  
- Instructional challenges and solutions  
- Measurable learning outcomes  

---

## Technologies Used

- **Amazon Bedrock** – Foundation models and generative AI  
- **Amazon SageMaker** – Model training, deployment, and experimentation  
- **AWS IAM** – Identity and access management  
- **EC2 / Cloud Infrastructure** – Lab environments  
- **Jupyter Notebooks** – Interactive ML workflows  
- **Hugging Face (Optional Integration)** – Model experimentation  

---

## Key Components

### 1. Curriculum Transformation
- Mapping AWS Academy content to AI/ML workflows  
- Integrating generative AI concepts into existing courses  
- Designing modular, scalable lab exercises  

### 2. Lab Architecture
- Cloud-based environments using AWS services  
- Sandbox and cost-controlled configurations  
- Multi-service integration (Bedrock + SageMaker)  

### 3. Security & IAM Design
- Role-based access for students  
- Service-level restrictions in learner labs  
- Secure access to AI services  

### 4. Student Projects
- AI-powered applications  
- Model experimentation and evaluation  
- Real-world use cases using Bedrock APIs  

### 5. Challenges Encountered
- Service availability in learner labs  
- Cost and resource limitations  
- Model access restrictions  
- Environment configuration complexity  

### 6. Student Outcomes
- Increased engagement with applied AI  
- Hands-on experience with industry tools  
- Improved understanding of ML workflows  
- Development of cloud-based AI projects

## Learning Outcomes

By implementing this framework, students are able to:

- Understand generative AI concepts in cloud environments  
- Build and test models using SageMaker  
- Use foundation models via Bedrock APIs  
- Design secure and scalable AI architectures  
- Deploy real-world AI applications
